# Deep Learning & Green AI: Classification and Regression

## 📌 Project Overview
This repository contains the final project for the **Selected Topics in Software Applications** course. It demonstrates the implementation of Deep Neural Networks (DNNs) using TensorFlow/Keras to solve two distinct problems, while actively tracking the environmental impact (Carbon Emissions & Energy Consumption) of the training process.

## 📂 Repository Structure
The project is divided into two main tracks:

### 1. Classification (Titanic Survival Prediction)
- **Objective:** Predict passenger survival (1 = Survived, 0 = Deceased).
- **Dataset:** `train.csv`
- **Model Architecture:** Dense layers (32 -> 16 -> 1) with `ReLU` and a final `Sigmoid` activation function.
- **Results:** Evaluated using Accuracy, F1-Score, and ROC AUC.

### 2. Regression (Melbourne Housing Price Prediction)
- **Objective:** Predict continuous real estate prices.
- **Dataset:** `melb_data.csv`
- **Model Architecture:** Dense layers (64 -> 32 -> 1) with `ReLU` and NO activation function in the output layer.
- **Results:** Evaluated using MAE, MAPE, RMSE, and $R^2$ Score.

## ⚙️ Instructions for Running the Project
To run the source code locally or on cloud platforms (like Google Colab or Kaggle):

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   ```
2. **Install Required Libraries:**
   ```bash
   pip install pandas numpy scikit-learn tensorflow matplotlib codecarbon
   ```
3. **Running the Notebooks:**
   - Open `.ipynb` files using Jupyter or Google Colab.
   - Ensure the `.csv` datasets are in the same directory.
   - Run all cells to execute training and evaluation.

## 🌍 Sustainability (Green AI)
Both notebooks utilize the `CodeCarbon` library to calculate energy consumed (kWh) and carbon emissions (kg CO2) generated by the hardware during training.

Trained `.h5` files are included for quick inference.
